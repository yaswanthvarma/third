c – create a archive file.
x – extract a archive file.
v – show the progress of archive file.
f – filename of archive file.
t – viewing content of archive file.


Create tar Archive File
tar -cvf test-2018-07-20.tar /home/test/


Create tar.gz Archive File
To create a compressed gzip archive file we use the option as z. 
tar cvzf MyImages-2018-07-20.tar.gz /home/MyImages

Untar tar Archive File

tar -xvf public_html-2018-07-20.tar
tar -xvf public_html-2018-07-20.tar -C /tmp/
tar xvf thumbnails-2018-07-20.tar.gz

List Content of tar Archive File

tar -tvf uploadprogress.tar
tar -tvf staging.test.com.tar.gz


Untar Single file from tar File

tar -xvf cleanfiles.sh.tar cleanfiles.sh
tar --extract --file=cleanfiles.sh.tar cleanfiles.sh
tar -zxvf testbackup.tar.gz testbackup.xml
tar -xvf test-2018-07-20.tar "file 1" "file 2"

Add Files or Directories to tar Archive File
tar -rvf test-2018-07-20.tar xyz.txt
